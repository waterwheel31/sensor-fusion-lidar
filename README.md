# Lidar Data Handling Excercise

This is an excersice of Lidar data handling, as a project of Udacity Sensor Fusion Nanodegree. 







### How to Run? 

- Install following 
    - cMake
    - PCL 
- Following code on terminal
    - mkdir build && cd build
    - cmake .. 
    - make 
    - ./environment 

    
### Environment 
- Tested on Ubuntu 18.04
